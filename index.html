List of mostly used HTML tags

  • <html> - содержит весь HTML документ
  • <head> - содержит ссылки на подключаемые файлов, в нём задаются параметры документа
  • <body> - содержит всё, что будет отображаться в браузере
  • <meta> - служит для задания различных параметров, например: название страницы, её описание, кодировка символов
  • <link> - подключает файлы стилей
  • <style> - содержит стили, применяемые к документу
  • <script> - содержит javascript код или подключает соответствующие файлы
  • <p> (paragraph) - текстовоый абзац
  • <div> - блок контента
  • <span> - содержит контент, который надо отобразить как-то особенно
  • <h1> - <h6> (headings) - для представления заголовков
  • <form> - для создания формы
  • <input> - текстовое поле
  • <textarea> - многострочное текстовое поле
  • <select> (<option>) - выпадающее меню
  • <br> - перевод строки
  • <a> - ссылка
  • <button> - для представления кнопки
  • <ul>, <ol>, <li> - списки и их элемент
  • <img> (image) - картинка
  • <svg> - представление векторной графики
  • <main>, <section>, <header>, <article>, <aside>, <footer>, <nav> - как DIV, только ещё описывают назначение блока
  • <video>, <audio> - для подключения видео и аудио на страницу
  • <iframe>, <embed> - для подключения на страницу блоков из других сайтов
  • <table>, <tr> <td> - для отображения таблицы
  • <strong>, <b>, <i> - для текста с оформлением
